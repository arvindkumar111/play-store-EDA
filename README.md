Playstore App Review Analysis
This project performs an in-depth Exploratory Data Analysis (EDA) on Playstore app reviews to uncover user sentiments, app performance trends, and key insights for app developers. Using Python, Pandas, and visualization libraries, we analyze 15 insightful plots to reveal patterns in ratings, review sentiment, app categories, and user feedback.
